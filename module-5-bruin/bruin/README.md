## Data Pipeline with Bruin
In this repository I've used a tool called [Bruin](https://github.com/bruin-data/bruin) to create an end-to-end data pipeline by utilizing MCP and a previously given template. 

The Pipeline used NYC Taxi database as instructed as part of the [module-5 of the DataTalks Data Engnieering Zoomcamp 2026](https://github.com/DataTalksClub/data-engineering-zoomcamp/tree/main/05-data-platforms). The relevant work can be seen in the 'zoomcamp' directory with the instructions given by the Bruin.
